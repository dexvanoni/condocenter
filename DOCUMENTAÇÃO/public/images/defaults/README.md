# Fotos Padrão dos Espaços

Este diretório contém as fotos padrão para os diferentes tipos de espaços.

## Tipos de Espaço:
- `party_hall.jpg` - Salão de Festas
- `bbq.jpg` - Churrasqueira
- `pool.jpg` - Piscina
- `sports_court.jpg` - Quadra Esportiva
- `gym.jpg` - Academia
- `meeting_room.jpg` - Sala de Reuniões
- `space.jpg` - Espaço Genérico

## Como adicionar fotos:
1. Adicione as imagens com os nomes corretos
2. Formatos aceitos: JPG, PNG, GIF
3. Tamanho recomendado: 800x600 pixels
4. Tamanho máximo: 1MB por imagem

## Nota:
Se uma foto não for encontrada, o sistema usará `space.jpg` como padrão.
